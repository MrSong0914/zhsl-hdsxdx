#  智慧水利-河道水下地形监测系统

#### 介绍
基于河道管理的实际需求，平台在场景数据可视化的基础上建设基础浏览、空间查询，数据分析与统计、场景三维仿真以及数据聚合展示等功能模块，各个模块基于凭条的统一数据处理机制下运行，并最终呈现到显示终端，为河道管理者在管理决策中提供提供真实直观有用的数据支持。
